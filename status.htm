Roadmap

Stage 1 - A helper program that allows users to locate the appropriate classification code for a particular topic category.

ccowhelp.py - A listing application (v1.00 - Jan 28, 2012)

Stage 2 - A program to faciliatate collaboration between experts in various fields to allow them to make suggestions for the remaining categories, vote on the suggestions, and finalize the results of those decisions for use by the helper application from Stage 1.

ccowprogress.py - Progress report generator (v1.00 - Feb 2, 2012)

ccoweditor.py - Suggestion functions

ccoweditor.py - Voting functions

ccoweditor.py - Approval functions

Stage 3 - Spider modules search engines can use to crawl the meta data proposed by this project, to compile trusted authority repositories that track false information or well-established sites, and to become trusted authorities themselves.

ccowcrawler.py - A spider module
